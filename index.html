1 Intro

1.1 Detahles PlatformIO CLI

Comandos pio = alias do comando platformio

  • pio platform install <platform> ::- Instala plataforma para projetos
  • pio project init ::- Inicia um novo projeto dentro da pasta atual
  • pio devices list ::- Lista dispositivos conectados, detectados pelo PlatformIO
  • pio lib -g install <library> ::- Instala a biblioteca em modo global do pio
  • pio lib search <library> ::- Procura pela lib.(Nome da lib entre aspas)
  • pio home ::- Inicia a home (Como nos IDE)
  • Comandos dentro do diretorio de projetos
    • pio run ::- Compila o código fora do device.
    • pio run --list-targets ::- Mostra os possíveis modos de compilação(compilar, limpar arquivos, upload para device…).
